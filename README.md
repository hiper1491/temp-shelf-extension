# 📚 TempShelf (臨時書架) V1.0

這是我第一個開發的 Chrome 擴充功能！解決了我書籤列總是被「購物暫存」和「技術資料」塞滿的痛點。

### 🌟 厲害的功能
- **一鍵打包**：按一下，所有分頁自動變成書籤，分頁列瞬間乾淨。
- **自動分類**：會自己掃描網址和標題關鍵字（支援 iHerb, 蝦皮, GitHub 等）。
- **保底收納**：不符合分類的網頁也會乖乖進到「其他」資料夾，絕對不漏抓。
- **雲端同步**：利用 Chrome 書籤同步，家裡存的，公司也能看。

### 🛠️ 如何安裝到 Chrome
1. 下載這個資料夾。
2. 打開 Chrome 的 `chrome://extensions/`。
3. 開啟右上角的「開發者模式」。
4. 點選「載入解壓縮擴充功能」，選擇這個資料夾。

### 📝 開發心得
- 學習了 JavaScript 的 `async/await` 非同步處理。
- 實作了關鍵字掃描邏輯，讓分類更聰明。
- 透過 VS Code 與 GitHub 進行版本管理。

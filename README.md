# 📚 TempShelf (臨時書架)

這是我專門為了拯救「爆炸的分頁列」而寫的 Chrome 擴充功能！

### 🌟 厲害的功能
- **一鍵打包**：按一下，所有分頁自動變成書籤，分頁列瞬間乾淨。
- **自動分類**：會自己認出「購物」還是「技術資料」。
- **重複排除**：存過的網頁就不會再重複存，省空間。
- **雲端同步**：利用 Chrome 書籤同步，家裡存的，公司也能看。

### 🛠️ 如何安裝
1. 下載這個資料夾。
2. 打開 Chrome 的 `chrome://extensions/`。
3. 開啟「開發者模式」。
4. 點選「載入解壓縮擴充功能」，選這個資料夾。
